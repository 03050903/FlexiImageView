FlexiImageView

FlexiImageView is a powerful extension of Android's default ImageView.

Download this project as a .zip file Download this project as a tar.gz file

Using the library

Adding the library to your project

You can download the library as a JAR file (recommended) or fork the entire project source into Android Studio. Once you've downloaded the library, add it to your project as a module.

Adding the view to your layouts

You can add FlexiImageView to any layout within an Activity or a Fragment. FlexiImageView is an extension of Android's built-in ImageView view, so you can add it to your XML files just like you'd add an ImageView: 

<com.psaravan.flexiimageview.lib.View.FlexiImageView
          android:id="@+id/flexiImageView"
          android:layout_height="wrap_content"
          android:layout_width="wrap_content" />

Using FlexiImageView's features

First, create an instance of FlexiImageView within the Activity or Fragment where you added the view: 

FlexiImageView flexiImageView = (FlexiImageView) findViewById(R.id.flexiImageView);

You can now use FlexiImageView's public methods to apply effects/customize your view. Here's an example that changes the view's shape to an oval: 

mFlexiImageView.setImageResource(R.drawable.test);
        
mFlexiImageView.setShape(FlexiImageView.SHAPE_OVAL)
               .setOvalHorizontalRadius(0.75f)
               .setOvalVerticalRadius(0.25f)
               .draw();

And here's the result:

Here are a few important points to note:

  • The 
    draw()
    method is what actually displays the image in the view container.
  • You must call setImageBitmap(), setImageResource(), or setImageDrawable() before calling the draw() method.
  • The draw() method should be the last method you call on your FlexiImageView instance. If you change any of the FlexiImageView's properties, you must call draw() again for the changes to be made visible.
